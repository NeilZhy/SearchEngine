Boost Exception tuple/operator<< include namespace boost { template E const & operator<<( E const & x, tuple< error_info, ..., error_info > const & v ); } Requirements: E must be boost::exception, or a type that derives (indirectly) from boost::exception. Effects: Equivalent to x << v.get<0>() << ... << v.get(). Returns: x. Throws: std::bad_alloc, or any exception emitted by T1..TN copy constructor.
