Author(s) Ron Garcia First Release 1.29.0 Standard Categories Containers, Math and numerics Dynamic Bitset The dynamic_bitset class represents a set of bits. It provides accesses to the value of individual bits via an operator[] and provides all of the bitwise operators that one can apply to builtin integers, such as operator& and operator<<. The number of bits in the set is specified at runtime via a parameter to the constructor of the dynamic_bitset.
