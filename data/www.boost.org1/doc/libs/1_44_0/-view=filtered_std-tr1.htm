Author(s) Greg Colvin, Beman Dawes, Peter Dimov, Darin Adler and Glen Fernandes First Release 1.23.0 Standard TR1 Categories Memory TR1 The TR1 library provides an implementation of the C++ Technical Report on Standard Library Extensions. This library does not itself implement the TR1 components, rather it's a thin wrapper that will include your standard library's TR1 implementation (if it has one), otherwise it will include the Boost Library equivalents, and import them into namespace std::tr1.
