Debug implementation does not have the same thread safety guarantees as the normal (release) implementation or other containers. As a result of this change BOOST_CB_DISABLE_DEBUG macro is no longer used. BOOST_CB_ENABLE_DEBUG=1 should be defined instead to enable debug support.
